Exterminador de Enlaces Rotos

Herramienta Profesional de QA para Auditar Sitios Web

🐍 Python ✅ QA Automation 📊 Reportes CSV/Excel 🎨 CLI Colorida

📋 ¿Qué hace esta herramienta?

El Exterminador de Enlaces Rotos es un script de Python que audita automáticamente todos los enlaces de un sitio web, detecta los que están rotos (404, 500, timeouts) y genera un reporte profesional en formato CSV listo para Excel.

Reporte profesional

✨ Características

🔍 Escaneo Automático

Extrae todos los enlaces <a href> de cualquier página web automáticamente

✅ Verificación de Status

Verifica códigos 200, 404, 500, timeouts, errores SSL y de conexión

🎨 Interfaz Colorida

Console output con colores: Verde para OK, Rojo para errores

🤖 Anti-Bloqueo

User-Agent falso para evitar que los sitios bloqueen el bot

📊 Reporte Excel

Genera CSV con URL_Origen, Link_Roto y Error para análisis

🚀 Rápido y Eficiente

Audita sitios completos en segundos sin intervención manual

🛠️ Stack Tecnológico

Python 3.x Requests BeautifulSoup4 Colorama Pandas

💻 Instalación

Clona el repositorio e instala las dependencias:

# Clonar el repositorio
git clone https://github.com/FerDVaz09/FerDVaz09.git
cd FerDVaz09/Exterminador-de-Enlaces-Rotos

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar el script
python broken_link_checker.py

🎮 Uso

Ejecuta el script y proporciona la URL a auditar:

python broken_link_checker.py

# Introduce la URL a auditar:
http://the-internet.herokuapp.com/status_codes

✅ [200 OK] https://github.com/...
❌ [404 NOT FOUND] http://example.com/broken
⚠️ [500] http://example.com/error

📄 Reporte guardado como: 'reporte_errores_qa.csv'

📊 Salida del Reporte

El archivo CSV generado incluye:

  • URL_Origen: La página auditada
  • Link_Roto: El enlace que falló
  • Error: Tipo de error (404, 500, Timeout, etc.)
