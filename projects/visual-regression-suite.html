🔧 ¿Qué es este proyecto?

Suite de pruebas de regresión visual en Python que detecta cambios visuales pixel a pixel. Utiliza Pillow para comparar capturas de referencia (baselines) contra ejecuciones actuales, calculando el porcentaje exacto de diferencia.

Las áreas con cambios se resaltan en rojo en el reporte HTML generado. Cubre 4 resoluciones de pantalla críticas (1920px, 1280px, 768px, 390px) para asegurar que el diseño responsive se mantenga intacto tras cada commit.

🎯 Características Principales

Snapshot Testing

Captura automática de screenshots de referencia y comparación instantánea contra ejecuciones futuras.

Multi-Resolution

Testing simultáneo en Desktop, Tablet y Mobile para prevenir "broken layouts" en dispositivos específicos.

Pixel Comparison

Algoritmo de comparación basado en Pillow que resalta diferencias mínimas de color o posición.

Allure Integration

Reportes profesionales con Allure que muestran la imagen esperada, la actual y el mapa de diferencias (diff).

Tolerance Control

Configuración de umbrales de tolerancia para ignorar cambios insignificantes causados por anti-aliasing.

CLI Interface

Interfaz de comandos para actualizar baselines o ejecutar auditorías específicas de forma rápida.

🧪 Stack Técnico Detallado

  • 🐍 Python + Pytest: Orquestador de pruebas y reporte de assertions.
  • 🎨 Pillow (PIL): Manipulación de imágenes y generación de Mapas de Calor de errores.
  • 🎭 Playwright Python: Motor de renderizado y captura de screenshots de alta calidad.
  • 🔗 CI/CD Pipelines: Integración con GitHub Actions para testeo automático en cada Pull Request.
