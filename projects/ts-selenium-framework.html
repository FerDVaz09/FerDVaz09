🔧 ¿Qué es este proyecto?

Framework de automatización web moderno construido completamente en TypeScript, que combina el poder de Selenium WebDriver con las ventajas del tipado estático para crear tests más robustos y fáciles de mantener. Diseñado desde cero con buenas prácticas de ingeniería de software aplicadas al QA.

La estructura del proyecto separa claramente el código fuente, los tests y la configuración, con un pipeline CI/CD mediante GitHub Actions que garantiza la ejecución automática de la suite en cada cambio. El tipado estricto de TypeScript previene errores en tiempo de compilación antes de que lleguen a ejecución.

🎯 Características Principales

Tipado Estático

TypeScript garantiza que los selectores, payloads y respuestas estén correctamente tipados, reduciendo bugs en tiempo de ejecución.

Selenium WebDriver

Automatización cross-browser completa para simular interacciones reales de usuario en múltiples entornos de navegación.

Estructura Modular

Organización en /src, /tests y /config que permite escalar el framework con nuevos módulos sin afectar el código existente.

CI/CD con GitHub Actions

Workflows configurados para ejecutar la suite de tests automáticamente en cada pull request, asegurando la calidad del código.

Gestión con npm

Dependencias y scripts centralizados en package.json para un setup rápido y reproducible en cualquier entorno de desarrollo.

Configuración Flexible

Directorio /config dedicado para manejar entornos, URLs base y opciones de WebDriver sin hardcodear valores en los tests.

🧪 Tecnologías Críticas

  • 📘 TypeScript: Tipado estricto para mayor robustez y detección temprana de errores.
  • 🌐 Selenium WebDriver: Automatización de navegadores para tests E2E de interfaz de usuario.
  • 📦 Node.js / npm: Entorno de ejecución y gestión de dependencias del ecosistema JavaScript moderno.
  • ⚙️ tsconfig.json: Configuración de compilación TypeScript para output optimizado y estricto.
  • 🏗️ GitHub Actions: Pipelines de CI/CD para ejecución automática en entornos de integración.
