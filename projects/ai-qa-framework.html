🔧 ¿Qué es este proyecto?

Framework de automatización de pruebas E2E de próxima generación que integra inteligencia artificial para generar test cases en segundos. Ingresa cualquier URL y Gemini AI analiza la interfaz y produce scripts de Playwright listos para ejecutar.

Incluye pipeline CI/CD con GitHub Actions que corre automáticamente en cada commit, generando reportes HTML con screenshots y videos de fallos. Arquitectura dockerizada garantiza ejecución consistente en cualquier entorno.

🎯 Características Principales

IA Generation

Uso de Gemini AI para analizar el DOM y generar selectores y flujos de prueba automáticamente.

Playwright Engine

Basado en el motor de Playwright para una ejecución ultrarrápida y soporte multiplataforma.

Dockerized

Entorno de ejecución aislado con Docker para evitar problemas de dependencias en diferentes sistemas.

Reports HTML

Reportes detallados con trazas de error, fotos y videos de cada ejecución fallida.

CI/CD Integration

Pipeline completo en GitHub Actions para ejecución en cada push y pull request.

Scale Ready

Diseñado para escalar de una sola prueba a cientos con orquestación eficiente.

🧪 Arquitectura Relevante

  • 🚀 IA Analyzer: Módulo que conecta con Gemini Pro API para entender la estructura de la página.
  • ⚙️ Test Generator: Motor que transforma el análisis en código Playwright (.spec.js).
  • 📦 Execution Engine: Ejecutor dockerizado que corre los tests en paralelo.
  • 📊 Report Dispatcher: Publicación automática de reportes en GitHub Pages o alertas vía Slack.
