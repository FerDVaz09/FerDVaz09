Exterminador de Enlaces Rotos

Python BeautifulSoup Requests Pandas Colorama

Script profesional de Python para realizar auditorías automáticas de calidad en sitios web, detectando enlaces rotos, URLs con errores y problemas de navegación que afectan la experiencia del usuario y el SEO.

Características Clave:

  • ✅ Escaneo Automático: Extrae y verifica todos los enlaces de una página sin intervención manual
  • ✅ Verificación de Status: Detecta códigos HTTP (200, 404, 500) y timeouts
  • ✅ Reportes CSV: Genera reportes profesionales con URL origen, enlaces rotos y códigos de error
  • ✅ Interfaz Visual: Feedback en tiempo real con colores (verde=OK, rojo=error)
  • ✅ User-Agent Falso: Simula navegador real para evitar bloqueos
  • ✅ Manejo Robusto: Gestión inteligente de errores y excepciones

Flujo de Testing:

  1. Extracción: BeautifulSoup parsea el HTML y extrae todos los enlaces
  2. Verificación: Requests comprueba cada enlace con headers personalizados
  3. Análisis: Clasifica los enlaces por status code (OK, Error, Timeout)
  4. Reporte: Genera CSV con enlaces rotos listos para revisión
  5. Presentación: Feedback visual inmediato en consola con colores

Casos de Uso en QA:

  • 🎯 Testing Pre-Producción: Validar que no hay enlaces rotos antes del lanzamiento
  • 🎯 Auditorías Periódicas: Verificación continua de sitios en producción
  • 🎯 Migración de Sitios: Confirmar que todos los enlaces apuntan correctamente
  • 🎯 SEO Quality: Asegurar que no hay errores 404 que afecten el ranking
  • 🎯 Reportes de Cliente: Documentación profesional de problemas encontrados

QA Fuzz Master

Python Flask Selenium Chrome Driver Faker

Aplicación web profesional para realizar pruebas de seguridad automatizadas mediante fuzzing, identificando vulnerabilidades como SQL Injection, Buffer Overflow y Cross-Site Scripting (XSS) en aplicaciones web.

Características Clave:

  • 🔒 SQL Injection Testing: Detecta vulnerabilidades con payloads como "' OR '1'='1"
  • 💥 Buffer Overflow: Prueba límites de validación con cadenas de 500+ caracteres
  • 🎭 XSS (Cross-Site Scripting): Inyecta scripts y caracteres especiales para probar filtros
  • 📸 Screenshots Automáticos: Captura evidencia visual antes y después de cada prueba
  • 🤖 Selenium WebDriver: Automatización completa de browser con Chrome headless
  • 🎨 Interfaz Cyberpunk: Diseño visual futurista con efectos glitch y animaciones

Flujo de Testing:

  1. Configuración: Usuario introduce URL objetivo y selecciona tipo de prueba
  2. Setup Selenium: Se inicializa Chrome Driver en modo headless
  3. Inyección de Payloads: Faker genera datos maliciosos específicos para cada test
  4. Ejecución: Selenium automatiza la interacción con formularios inyectando payloads
  5. Captura: Screenshots automáticos documentan el comportamiento del sistema
  6. Reporte: Resultados en tabla con evidencias visuales y estado de cada prueba

Tipos de Ataques Simulados:

🔐 SQL Injection:

  • Payloads: ' OR '1'='1, admin'--, '; DROP TABLE users--
  • Objetivo: Bypasear autenticación y acceso a datos no autorizados

💥 Buffer Overflow:

  • Genera cadenas de 500+ caracteres aleatorios
  • Objetivo: Probar validación de longitud máxima en campos

🎭 Cross-Site Scripting (XSS):

  • Payloads: <script>alert('XSS')</script>, emojis especiales 💀🔥💻
  • Objetivo: Inyectar código JavaScript y caracteres Unicode maliciosos

⚠️ Uso Ético y Responsable

Esta herramienta está diseñada exclusivamente para testing autorizado en entornos controlados. Solo debe usarse en aplicaciones para las que se tenga permiso explícito. El uso indebido puede violar leyes locales e internacionales de ciberseguridad.

QA Log Sentinel

Python Flask HTML5 / CSS3 JavaScript Fetch API

Sistema profesional de monitoreo visual de logs en tiempo real con interfaz tipo terminal/hacker inspirada en Matrix. Simula y visualiza eventos del sistema como errores críticos, advertencias y mensajes informativos con actualización automática cada 2 segundos.

Características Clave:

  • 📊 Monitoreo en Tiempo Real: Polling automático cada 2 segundos sin WebSockets
  • 🎨 Interfaz Matrix: Diseño CRT con efectos de scanlines, flicker y glow verde fosforescente
  • 🚨 Simulación de Eventos: Genera errores críticos, advertencias y mensajes INFO con IDs únicos
  • 🎯 Colores Diferenciados: Rojo para [ERROR], Naranja para [WARNING], Cyan para [INFO]
  • 📈 Estadísticas Visuales: Panel de métricas actualizado en tiempo real
  • 🔄 Auto-scroll Inteligente: Siempre muestra la actividad más reciente

Flujo de Funcionamiento:

  1. Inicialización: Flask crea archivo server.log si no existe
  2. Polling: Cliente consulta /get_logs cada 2 segundos
  3. Simulación: Usuario genera eventos mediante botones del panel de control
  4. Escritura: Backend escribe en server.log con timestamps y niveles
  5. Renderizado: JavaScript aplica colores y efectos según tipo de log
  6. Métricas: Contadores actualizados automáticamente (Total, Errors, Warnings, Info)

Tipos de Mensajes Generados:

🔴 Errores Críticos [ERROR]:

  • Critical Database Connection Failed #DB1234
  • Memory Overflow Detected #MEM5678
  • API Authentication Failed #AUTH9012
  • SSL Certificate Validation Failed #SSL2345

🟠 Advertencias [WARNING]:

  • High CPU usage detected (85%)
  • Low memory available (150MB free)
  • Slow database query detected (5.2s)
  • Rate limit approaching threshold

🔵 Información [INFO]:

  • User login successful
  • Data backup completed
  • Database sync completed
  • Health check passed

🎨 Efectos Visuales Especiales

La interfaz incluye efectos únicos tipo terminal: Scanlines CRT (líneas horizontales animadas), Flicker Effect (parpadeo sutil), Text Glow (resplandor verde fosforescente), y Hover Animations con transiciones suaves. El scrollbar está personalizado con colores Matrix y las notificaciones usan toast messages animadas.
