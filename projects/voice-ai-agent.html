🔧 ¿Qué es este proyecto?

Sofia es un agente de voz autónomo diseñado para entornos de salud. Gestiona llamadas telefónicas entrantes y salientes, mantiene conversaciones naturales en tiempo real y agenda citas médicas directamente en la base de datos, sin intervención humana.

Construido con FastAPI como backend, utiliza Vapi.ai para la gestión de llamadas, Deepgram para transcripción de voz a texto, ElevenLabs para síntesis de voz natural y Groq Llama 3 70B como modelo de lenguaje principal por su velocidad y eficiencia de costo. Todo orquestado con Docker Compose y respaldado por PostgreSQL 16.

🎯 Características Principales

Llamadas Autónomas

Gestión completa de llamadas entrantes y salientes vía Vapi.ai sin intervención humana, disponible 24/7 para pacientes.

Groq Llama 3 70B

Modelo de lenguaje de alta velocidad para respuestas naturales y coherentes durante conversaciones médicas en tiempo real.

Agendamiento de Citas

Herramientas custom con integración directa a base de datos PostgreSQL para consultar disponibilidad y registrar citas automáticamente.

Voz Ultra-Realista

ElevenLabs genera síntesis de voz de alta calidad y Deepgram convierte la voz del paciente a texto con gran precisión.

Webhooks en Tiempo Real

FastAPI procesa eventos de llamada al instante mediante webhooks, manteniendo el flujo conversacional sin latencia perceptible.

Infraestructura Containerizada

Docker Compose orquesta la base de datos PostgreSQL y el servidor para un despliegue reproducible en cualquier entorno.

🧪 Tecnologías Críticas

  • ⚡ FastAPI (Python): Backend de alto rendimiento con webhooks para procesamiento de eventos en tiempo real.
  • 📞 Vapi.ai: Plataforma de telefonía IA para gestión de llamadas entrantes y salientes.
  • 🎙️ Deepgram: Speech-to-text de alta precisión para transcribir la voz del paciente.
  • 🔊 ElevenLabs: Text-to-speech ultra-realista para la voz de Sofia.
  • 🤖 Groq Llama 3 70B: Modelo LLM de inferencia rápida para respuestas conversacionales naturales.
  • 🗄️ PostgreSQL 16: Base de datos relacional para registro de interacciones y citas médicas.
  • 🐳 Docker Compose: Orquestación de servicios para despliegue consistente y portable.
